# Interface: RewardsDataPoint

Single atomic rewards point for a particular date

## Properties

### when

• **when**: `Date`

**`Const`**

A reference date for the rewards received

___

### amount

• **amount**: `bigint`

**`Const`**

Amount of rewards received in gwei

___

### vault

• **vault**: \`0x$\{string}\`

**`Const`**

Address of the vault that generated the rewards
